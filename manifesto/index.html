2. Tools

Heroku

Continuous deployment platform and developer marketplace.

MongoDB

Big data storage with Javascript under the hood.

Node.js

Shared client/server application logic, with non-blocking, asynchronous design for high throughput.

Derby.js

Realtime, collaborative applications that run in both Node.js and browsers.

HTML 5

Connectivity, storage, performance, graphics, audio, video, and so much more.

Phonegap

A platform for deploying HTML5 apps to multiple app stores.

Github

Source code hosting, wiki documentation, and issue tracking in a single package, all powered by git.

3. Products

Developer APIs

A structured collection of data about all the things in our world and the relationships between them, distributed via Node.js over swaggering APIs, with a scaling price structure.

Mobile Apps

To help prove and improve our own APIs, we build apps. Some cost money. Some are free. All of them give us valuable data about user behavior, and they feed the word graph.

Own-a-Word

In the spirit of community, wordnik.com users will be given a socially incentivized option to provide financial support by sponsoring words of their choice, with highest-bidder game logic to promote healthy competition for word "ownership".

4. Principles

Transparency

Open source software promotes trust from users and crowdsources the development effort to a willing constituency.

Community

Giving end-users more control encourages their level of participation. Continued development of Swagger and other open-source projects earns us respect in the developer community.

Agility

We are a small, passionate team, free from the constraints of hierarchy and creative mediocrity, with fewer bottlenecks and higher output.

Passion

We love the work we do: We dream about it, talk to our families about it, and devote our minds to it.
We create beautiful things.
